# Design Notes

- The current `Choose Project` screen is the approved visual reference.
- Keep the current navbar/top bar direction.
- Keep the current card layout and spacing.
- Keep the current soft white background with light green and light red tint.

## Why This Works
- Feels cleaner and more premium than the overly plain variant.
- Navbar now feels more aligned with the `Slip` page.
- Background is softer and more brand-friendly without becoming noisy.

- Reuse the same navbar language from `Choose Project` on the `Slip` page.

- Keep interactions clear and operational, but let the visual tone match this project picker.
- Do not over-redesign the `Slip` workflow; mainly align tone, top bar, spacing rhythm, and surface styling.

## Guardrail
- If revisiting `Choose Project`, avoid changing the main body/cards again unless truly necessary.
